# Point of Sale (POS) System

A full-stack Point of Sale (POS) system built with Django designed to help businesses manage sales transactions, products, and customers efficiently.

This project demonstrates backend architecture, payment integration, and role-based access control in a real-world business system.

---

## Features

- Sales and order processing
- Product and inventory management
- Customer management
- Role-based dashboards (Admin, Manager, Cashier)
- Receipt generation
- Transaction history tracking
- Mobile Money payment integration
- Clean and structured admin interface

---

## Tech Stack

Backend:
- Python
- Django

Frontend:
- HTML
- CSS
- JavaScript

Database:
- SQLite (development)

Payment Integration:
- Mobile Money (MoMo)

---

## System Roles

### Admin
- Manage products and inventory
- View sales reports
- Manage users

### Manager
- Monitor transactions
- View analytics and reports

### Cashier
- Process customer purchases
- Generate receipts
- Handle payments

---

## Project Goals

This project was built to simulate a real-world retail transaction system while demonstrating:

- Full-stack web development
- Backend logic implementation
- Database management
- Payment gateway integration
- Role-based authentication systems
